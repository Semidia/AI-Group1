# 🎮 凡墙皆是门 (Every Wall Is A Door) - 玩家手册

欢迎来到《凡墙皆是门》！这是一款模拟 CEO 决策的文字策略游戏。你的目标是带领“Nexus Corp”公司在商业竞争中生存下去。

## 🎯 核心目标
**保持生存！**
游戏界面顶部有四个关键属性，任何一个归零都意味着游戏结束（GameOver）：
- 💰 **现金 (Cash)**: 公司的生命线。
- 🎭 **士气 (Morale)**: 员工的积极性。
- 🌟 **声望 (Reputation)**: 市场的认可度。
- 💡 **创新 (Innovation)**: 持续发展的动力。

## 🕹️ 游戏模式
为了方便不同场景的体验，V1.1 版本引入了双模式：

### 1. 🧪 演示模式 (Demo Mode)
- **无需连接**：纯前端运行，无需启动后端服务器。
- **即开即玩**：适合快速体验游戏界面和基础流程。
- **丰富背景故事**：内置完整的预制背景故事，包括公司背景、当前困境、行业环境和玩家任务，增强游戏代入感。
- **注意**：此模式下的逻辑是简化的，且刷新页面会丢失进度。

### 2. 🚀 正式模式 (Official Mode)
- **完整体验**：连接强大的 Python 后端引擎（已接入真实大模型API）。
- **初始化设定**：可自定义游戏规则和AI行为要求，生成个性化背景故事
- **多种连接**：
    - **📡 本地部署 (Localhost)**: 适合开发者。需运行 `一键启动游戏.bat` 或在后台运行 `python main.py`。
    - **☁️ 远程调用 (Remote API)**: 连接我们部署在阿里云香港的服务器，享受最佳体验！
        - **无需配置网络**：玩家无需任何特殊网络设置即可使用
        - **稳定快速**：利用香港服务器的网络优势，流畅访问国际大模型API
        - **即连即玩**：输入服务器IP即可体验真实AI的强大能力
        - **推荐使用**：这是我们为玩家准备的最佳体验方案

## 🎮 操作指南

### 1. 启动游戏
- 双击项目根目录下的 **`一键启动游戏.bat`**。
- 浏览器自动打开后，在首页选择想要体验的模式。

### 2. 游戏界面
- **❌ 返回首页**: 界面左上角新增返回按钮，可随时切换模式。
- **❓ 帮助功能**: 界面右上角点击“帮助”可查看简易说明。
- **📜 剧情终端**: 中间黑框显示所有的历史决策和事件反馈。

### 3. 初始化设定 (正式模式)
- **访问方式**：在正式模式页面，点击“⚙️ 初始化设定”按钮。
- **功能**：自定义游戏规则和 AI 行为要求，生成个性化背景故事。
- **操作步骤**：
  1. 在模态框中输入你的自定义设定（可使用默认模板）。
  2. 点击“保存设定”后返回。
  3. 选择连接方式（本地部署或远程调用），AI 将根据你的设定生成背景故事。

### 4. 开始决策
现在采用多选+自定义输入+手动确认的分步交互流程，你可以：

#### 方式一：多选决策建议选项
1. **选择选项**：点击任意决策建议选项，可多次点击切换选中/取消选中状态
   - 选中的选项会显示高亮背景和勾选标记
   - 可同时选择多个选项
2. **查看选中状态**：已选选项的状态清晰可辨
3. **输入自定义内容**：可在下方输入框中补充自定义决策内容（可选）
4. **提交决策**：点击“执行”按钮，提交所有选中的选项和自定义输入

#### 方式二：仅输入自定义指令
1. **输入自定义内容**：在下方输入框键入你的想法（如“裁掉一半员工”、“投资比特币”）
2. **提交决策**：点击“执行”按钮

> *注：在正式模式下，系统会尝试理解你的自然语言指令。*
> *提示：未选择任何选项且未输入自定义内容时，会显示提示信息。*
