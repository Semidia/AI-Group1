# 📖 玩家手册

## 🎯 目标
保持公司生存！确保顶部四个状态条（现金、士气、声望、创新）不归零。

## 🎮 玩法
1. **听取建议**: 点击底部 AI 推荐的按钮（如“紧急融资”）。
2. **下达指令**: 在输入框键入你的决策（如“裁员”、“加大研发”），然后点击执行。

---

⚠️ **注意**: 这里是演习模式，刷新页面会丢失进度。您可以随时滚动查看历史记录。

## 玩家请忽视下面内容
只需 启动一次命令 ：
# 从根目录运行
node sync-help-content.js --watch
# 或使用 npm 脚本（根目录或前端目录均
可）
npm run sync-help:watch

这时在 github小组开发库说明书/帮助按钮里的内容.md 修改内容
保存文件后（Ctrl+S 或编辑器自动保存）
无需任何手动操作，系统会 自动检测修改 并 自动同步 到游戏中
