# 第三阶段测试（房间系统基础）

## 测试目标
- 覆盖房间基础流：创建 → 列表 → 加入 → 离开。
- 校验认证链路与房间接口契约是否符合第三阶段要求。

## 目录
- `test-phase3.ps1`：PowerShell 自动化测试脚本

## 运行前提
1. 后端已启动：`cd 重新打造/正式搭建/backend && npm run dev`
2. 数据库已就绪：PostgreSQL 运行且已执行 `npm run prisma:migrate`
3. （可选）Redis 运行，用于房间状态缓存

## 运行方法（Windows PowerShell）
```powershell
cd "重新打造/各阶段测试用/第三阶段测试"
.\test-phase3.ps1
```

## 预期验证点
- 健康检查通过
- 注册/登录 2 个用户成功
- 创建房间返回 room_id
- 房间列表返回包含新建房间
- 第二个用户加入/离开房间成功

## 常见失败原因
- 后端未启动或端口被占用
- 数据库未迁移 / 未运行导致 500
- 房间接口未实现返回 404/501

## 说明
若当前接口尚未完成，脚本会在对应步骤标红提示，可据此对齐开发任务。完成开发后再次运行脚本进行回归。
