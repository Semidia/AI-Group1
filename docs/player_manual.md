# 🎮 玩家操作手册

欢迎来到 **AI-Group1 智能体博弈游戏**！本作为一款基于大语言模型的多人联机策略游戏，你将与其他玩家和 AI 共同在动态演变的场景中进行竞争或合作。

## 1. 快速入门

### 1注册与登录
1.  打开游戏网址（通常由主持人提供）。
2.  点击 **"去注册"**。
3.  **极简注册**：仅需输入 **用户名** 和 **密码** 即可完成注册。
    *   *注意：为了简化流程，系统不再需要邮箱验证。*
4.  输入账号密码完成登录。

### 加入房间
1.  登录后，你将看到 **游戏大厅**，列表中展示了当前活跃的房间。
2.  找到主持人告知的房间号或房间名，点击 **"加入"**。
3.  如果房间设有密码，请输入主持人提供的密码。
4.  加入成功后，你会进入 **等候大厅**，请耐心等待所有玩家到齐及主持人开始游戏。

---

## 2. 游戏流程

游戏按 **回合制** 进行，每个回合通常包含以下阶段：

### ⏳ 决策阶段 (Decision Phase)
这是你发挥策略的主要时刻：
-   **查看情报**：阅读左侧的“当前局势”或“上一轮结果”，了解环境变化。
-   **提交行动**：在底部的输入框中描述你的行动计划。
    -   *提示：你可以尝试自由输入，比如“联合 Player 2 抬高价格”或“秘密通过小路运输物资”。AI 会解析你的意图。*
-   **发起交易**：点击 **"交易"** 标签页，可以选择向其他玩家发送资源交换请求。

### ⏸️ 评审阶段 (Review Phase)
-   主持人可能会在该阶段暂停，审查所有玩家提交的决策，确保没有违规操作。
-   此时你无法修改指令，只需等待。

### 🤖 推演阶段 (Inference Phase)
-   AI 正在根据通过的规则和所有玩家的行动计算结果。这是一个复杂的生成过程，请稍候。

### 📊 结果阶段 (Result Phase)
-   新回合开始前，系统会展示上一轮的 **推演结果**。
-   查看你的资源变化、获得的积分以及发生的随机事件。

---

## 3. 常见功能

-   **查看状态**：顶部状态栏实时显示你的 HP（生命值/存活度）、金币或其他资源。
-   **聊天/日志**：部分模式下支持公屏聊天或查看历史推演日志。
-   **退出房间**：如果你需要中途离开，可以点击右上角的退出按钮，但请注意这可能会影响游戏平衡。
