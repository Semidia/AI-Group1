# 默认规则查看功能说明

## 功能概述

在主持人配置页面的"Step 2 - 规则与玩家配置"部分，新增了"查看默认蓝本规则"功能，让主持人可以方便地查看和使用《凡墙皆是门》游戏的默认规则。

## 功能特性

### 1. 查看默认规则 📖
- 点击"查看默认蓝本规则"按钮打开规则查看窗口
- 规则内容经过格式化处理，易于阅读
- 支持标题层级显示，结构清晰

### 2. 复制规则 📋
- 一键复制完整规则到剪贴板
- 方便在其他地方使用或修改

### 3. 直接使用 ✨
- 点击"使用此规则"按钮直接填入表单
- 自动填充到游戏规则输入框
- 提醒用户保存配置

### 4. 智能提示 💡
- 规则输入框下方提供详细说明
- 解释留空的默认行为
- 提供Markdown格式支持说明

## 技术实现

### 前端组件
- `HostSetup.tsx` - 主配置页面
- `defaultRules.ts` - 规则常量定义
- Modal组件 - 规则查看窗口

### 功能特点
- 响应式设计，适配不同屏幕
- 格式化显示，支持Markdown样式
- 一键操作，用户体验友好

## 使用步骤

1. **进入配置页面**
   - 创建房间后进入主持人配置
   - 导航到"Step 2 - 规则与玩家配置"

2. **查看默认规则**
   - 点击"查看默认蓝本规则"按钮
   - 在弹出窗口中浏览完整规则

3. **选择使用方式**
   - **直接使用**: 点击"使用此规则"按钮
   - **复制修改**: 点击"复制规则"后自行修改
   - **留空使用**: 不填写规则，系统自动使用默认规则

4. **保存配置**
   - 点击"保存规则"按钮
   - 继续完成其他配置步骤

## 注意事项

- 默认规则与后端AI服务中的规则保持一致
- 留空规则配置将自动使用默认蓝本
- 支持Markdown格式，可以自定义样式
- 建议根据参与者水平适当调整规则复杂度
